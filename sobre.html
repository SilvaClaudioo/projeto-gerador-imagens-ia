Descrição do projeto

Este projeto é um gerador de imagens utilizando inteligência artificial. O usuário pode descrever a imagem que deseja e o sistema irá gerar uma imagem correspondente.

Tecnologias utilizadas

HTML

CSS

Objetivo do projeto

O objetivo deste projeto é demonstrar como a inteligência artificial pode ser utilizada para gerar imagens a partir de descrições textuais, proporcionando uma nova forma de criação artística.
